Hello, this is my first pet project in web development. The idea is simple: to create my own convenient and stylish browser start page with URL bookmark saving functionality. For this page, I used only HTML, CSS, and JavaScript, without frameworks, libraries, or style preprocessors. To save information between sessions, I decided to use localStorage. When creating a bookmark on the page, we will access this API, create or edit the "storageObjectOfMyApp" object (I added a function that outputs localStorage to the console when the site loads), and create the necessary number of bookmarks on the page based on this data. To create the background animation, I used an asynchronous function that calls itself with a delay, creates a column of numbers (randomly either 0 or 1), and deletes it with a delay as well. All the other 'magic' is done by CSS. Also, the page is adapted for different devices
